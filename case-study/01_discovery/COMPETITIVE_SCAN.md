# Competitive Scan — Digital Board Games

> **Purpose**: Understand the competitive landscape for online Catan-style games  
> **Note**: This is a high-level overview for portfolio context, not a comprehensive market analysis

---

## 1. Direct Competitors

### 1.1 Catan Universe (Official)
**Platform**: Web, Mobile, Desktop  
**Model**: F2P with expansion purchases

| Aspect | Notes |
|--------|-------|
| **Strengths** | Official license, polished visuals, campaign mode |
| **Weaknesses** | Slower pace, less accessible matchmaking |
| **UX Notes** | More traditional board game aesthetic |

### 1.2 Board Game Arena
**Platform**: Web  
**Model**: F2P + Premium subscription

| Aspect | Notes |
|--------|-------|
| **Strengths** | 800+ games, large community, async play |
| **Weaknesses** | Generic UI, not Catan-focused |
| **UX Notes** | Functional but dated interface |

### 1.3 Tabletopia / Tabletop Simulator
**Platform**: Web / Steam  
**Model**: Game purchases + DLC

| Aspect | Notes |
|--------|-------|
| **Strengths** | Physical simulation, extensive library |
| **Weaknesses** | Steep learning curve, requires setup |
| **UX Notes** | Physics-based, not streamlined |

## 3. UX Patterns to Learn From

### 3.1 Chess.com — Onboarding
- Clear skill-based pathways
- Interactive tutorials integrated into first games
- Progress tracking for beginners

### 3.2 Lichess — Simplicity
- Minimal UI, maximum focus on game
- One-click play options
- Clean, accessible design

### 3.3 Discord — Social Integration
- Friend presence (online/offline/in-game)
- Quick invite via link sharing
- Activity feed and notifications

### 3.4 Duolingo — Progressive Disclosure
- Simple start, complexity revealed over time
- Clear progress indicators
- Encouraging, friendly tone

## 4. Opportunity Areas for Colonist

Based on competitive analysis:

1. **Onboarding**: Most competitors lack great onboarding — opportunity to lead
2. **Speed**: Colonist already fast — maintain/enhance this advantage
3. **Social**: Discord-like presence indicators would improve friend play
4. **Clarity**: Simplify UI to match Lichess-level focus
5. **Membership**: Make value prop clearer than competitors
