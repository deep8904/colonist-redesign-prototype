# Wireframes Plan
## Colonist.io Redesign

## 1. Wireframe Strategy

### Approach
- **Fidelity**: Low-to-mid fidelity wireframes
- **Focus**: Layout, hierarchy, and user flow â€” not visual polish
- **Format**: Screen-by-screen with annotations
- **Tool**: Figma (see [FIGMA_PROMPT.md](../05_handoff/FIGMA_PROMPT.md))

### Wireframe Principles
1. Content hierarchy before aesthetics
2. Show real-ish content, not lorem ipsum
3. Annotate interaction patterns
4. Include responsive considerations
5. Group by user flow, not by page type

## 4. Responsive Considerations

### 4.1 Breakpoints

| Breakpoint | Width | Layout Notes |
|------------|-------|--------------|
| Mobile | < 640px | Single column, bottom tabs |
| Tablet | 640-1024px | 2 columns, side nav option |
| Desktop | > 1024px | Full layout, persistent nav |

### 4.2 Mobile-Specific Patterns

| Pattern | Desktop | Mobile |
|---------|---------|--------|
| Navigation | Top bar | Bottom tabs + drawer |
| Room cards | Side-by-side | Stacked |
| Filters | Visible bar | Collapsible |
| Friends list | Sidebar | Full-screen modal |
| Create wizard | Side panel | Full-screen steps |
